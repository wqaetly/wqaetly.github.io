《奇门遁甲》学习总结

奇门遁甲：起源、发展与体系精讲

一、 核心概念与名称释义

1. 奇门遁甲的定义

奇门遁甲是“三式”（太乙、奇门、六壬）之一。

  • 太乙神数：占天时、国运（天）。
  • 奇门遁甲：占地利、行军打仗、兵法运筹（地）。
  • 大六壬：占人和、日常百事（人）。
  • 地位：奇门最初是为军事战争而设，后演变为涵盖万事万物的预测学，被称为“三式之首”（虽有流派之争，但其兵家属性确凿）。

2. “奇门遁甲”四字详解

  • 奇（三奇）：即十天干中的 乙、丙、丁。
  • 门（八门）：指 休、生、伤、杜、景、死、惊、开（兼指九星）。
  • 遁（隐遁）：隐藏、保护。
  • 甲（六甲）：即十天干之首 甲（甲子、甲戌、甲申、甲午、甲辰、甲寅）。
    • 核心逻辑（甲庚之争）：
      • 矛盾根源：五行中，庚（阳金） 克 甲（阳木），且是强克（如斧劈木）。甲是统帅/元帅，不能轻易阵亡，必须隐藏起来（遁）。
      • 化解手段（三奇的作用）：
        1. 乙（乙庚合）：乙是甲的妹妹（阴木），将乙嫁给庚，以柔克刚，化敌为亲（和局）。
        2. 丙/丁（火克金）：木生火，丙丁是甲的子孙。用儿女（火）去克制敌人庚（金），使其不敢妄动（制局）。
  • 结论：奇门遁甲的本质是以三奇（乙丙丁）、八门、九星、六仪构建阵法，将甲隐遁起来，以抵御庚金的克制。

二、 基础符号体系

学习奇门遁甲必须掌握的底层积木：

  1. 阴阳五行：玄学的核心根基。阴阳是对立统一，五行（金木水火土）是生克流转。这是最高阶的技法心法。
  2. 十天干：甲、乙、丙、丁、戊、己、庚、辛、壬、癸。
    • 三奇：乙、丙、丁。
    • 六仪：戊、己、庚、辛、壬、癸（甲就隐遁在六仪之下）。
  3. 十二地支：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥（对应生肖与时间）。
  4. 九宫八卦：
    • 九宫：源于《洛书》，戴九履一，左三右七，二四为肩，六八为足，五居中宫（纵横斜和皆为15）。
    • 八卦：使用后天八卦（坎一、坤二、震三、巽四、乾六、兑七、艮八、离九）。
  5. 九星（天时）：天蓬、天任、天冲、天辅、天英、天芮（含天禽）、天柱、天心。
  6. 八门（人和）：休、生、伤、杜、景、死、惊、开。
  7. 八神（神助）：值符、腾蛇、太阴、六合、白虎、玄武、九地、九天。

三、 起源传说与数理演变

1. 神话传说

相传黄帝战蚩尤于逐鹿，久攻不下。九天玄女授黄帝“龙甲神章”，风后将其演译为文字，即奇门遁甲。

2. 局数的演变（从繁至简的数学逻辑）

奇门局数的演变并非失传，而是归纳总结的过程：

  • 第一阶段（风后演制）：4320局
    • 算法：一年360天 × 12个时辰 = 4320个时辰。每时辰一局，故共4320局。
  • 第二阶段（姜子牙/风后）：1080局
    • 算法：4320局中，每4局的盘面其实是重复的。4320 ÷ 4 = 1080局。
    • 结构：分为阳遁540局，阴遁540局。这是奇门真正不同的盘面总数。
  • 第三阶段（姜子牙/太公）：72局
    • 算法：一年24节气，每节气15天，分上、中、下三元（一元5天），共72候（元）。此为“72活局”。
  • 第四阶段（张良）：18局（现行通用）
    • 算法：将1080局归类，阳遁9局，阴遁9局，共18局。这是目前最常用的起局定式（即阳一局、阳二局…阴九局等）。

3. 历史文献考证

  • 东汉：《后汉书》首次出现“遁甲”之名，标志着术数成型。
  • 隋唐：官方严禁私习玄学（特别是涉及军事的三式），但在民间暗中流传发展。
  • 宋代（巅峰期）：宋仁宗编修**《景佑遁甲符应经》，确立了现代奇门遁甲的标准范本。《烟波钓叟歌》**（相传赵普著）也是此时期的重要著作。
  • 明清：流传广泛，著作频出。重要典籍包括**《奇门遁甲秘笈大全》（刘伯温）、《御定奇门宝鉴》**（康熙，俗称大宝鉴）、《遁甲演义》。

四、 奇门遁甲的分类

1. 按功能分

  • 数理奇门：基于易理、九宫八卦进行推演预测（本课程所学）。
  • 法术奇门：涉及符咒、修炼等内容（不在此课程范围内）。

2. 按时间单位分（四柱）

  • 年家奇门、月家奇门：用于断大事、长远事。
  • 日家奇门：有研究价值。
  • 时家奇门：最主流、最常用，精准度最高，宋代以来一直沿用。

3. 按起局（定局）方法分

  • 置闰法：古法，较复杂，讲究节气超接置闰。
  • 拆补法：本课程采用的方法。简便实用，直接依据当前节气和干支拆补，无需置闰。
  • 茅山道人法：结合二者。
  • 注：奇门无定法，不同流派只要逻辑自洽皆可，现代多用拆补法。

4. 按排盘方式分

  • 转盘奇门：本课程所学。九星、八门、八神在盘面上做圆周旋转。
  • 飞盘奇门：九星八门按照洛书九宫轨迹飞布（跳跃式）。
  • 注：转盘包含了“转”与“飞”的结合（因为九宫底盘本身就是飞布的洛书），体系更为完备。

五、 预测守则与学习心法

1. 预测师五戒（行规）

  1. 断后不测：不预言别人无后代、绝后。
  2. 死亡不测：不具体断言人的死亡时间（哪怕看出来了）。
  3. 赌博不测：彩票、赌博等投机之事测不准，且损耗极大。
  4. 谋私不测：不可利用奇门术数去行骗色、骗财等不义之事。
  5. 必收卦金：“值符”即钱财，收钱是为了了结因果。白嫖会带来无穷的麻烦和精力损耗，哪怕收一分钱也要收。

2. 学习建议

  • 重基础：阴阳五行和天干地支是重中之重。技法（招式）可以学几百条，但最后断局准不准，全看对阴阳五行的理解深度。
  • 多实战：看案例，要有自己的思考。别人的总结（技法）是别人的鞋，只有自己通过案例磨出来的思路才是自己的脚。
  • 核心认知：学习的是“奇门遁甲”，不是单学“奇门”。要理解其背后的军事运筹逻辑（甲庚矛盾），而非死记硬背符号。

气化阴阳：从本源到奇门实战应用总结

一、 阴阳的起源与概念

1. 气的本质

  • 字源释义： 依据许慎《说文解字》，“气”为“云气也”，是象形字。甲骨文中象征河两岸与干涸的河床，后演变为指代云气。
  • 特性： 气具有流动性和可知可感但不可触的特性（如热浪、寒气）。它是构成万物的基本能量，通过流动分化为阴阳。

2. 阴阳的诞生（从天象到地理）

  • 观测基础： 阴阳概念并非凭空产生的抽象哲学，而是古人对天象（日月运行、四季轮替）和物象（地理环境）最直观的观测记录。
  • 原始含义：
    • 阳： 云开雾散，日照当头，向日有光；山南水北。
    • 阴： 云遮蔽日，背日无光；山北水南。
  • 哲学升华： 从具体的光照、方位，演化为描述事物属性的对立统一概念（如动静、冷热、刚柔）。

3. 历史演变脉络

  • 夏商时期： 侧重于太阳出没的自然天象。
  • 西周时期： 结合《周易》，用阴阳爻作为符号；开始将阴阳视为“二气”，用阴阳二气的秩序失衡解释自然灾害（如伯阳父论地震：阳伏而不能出，阴迫而不能蒸）。
  • 春秋战国（老子）： “万物负阴而抱阳”，阴阳成为万物生成的内在根源。
  • 秦汉时期： 阴阳学说与五行、宗法制度结合，形成庞大的哲学体系（如《黄帝内经》、《白虎通义》）。

二、 阴阳十大定理（核心逻辑）

阴阳并非孤立存在，而是遵循特定规律：

  1. 相对性： 阴阳不是绝对的，而是比较而来的（如：男人相对于女人是阳，但相对于更高大的男人可能显阴；台灯相对于暗处是阳，相对于太阳是阴）。
  2. 互根互用： 阴阳相互依存，互为存在的根基（太极图中阴中有阳，阳中有阴）。
  3. 消长平衡： 阴消阳长，阳消阴长。如冬至一阳生，夏至一阴生。
  4. 对立统一： 阴阳性质相反（动与静、热与冷），但最终追求的是平衡与和谐，而非一方消灭另一方。
  5. 其他性质： 包括阴差阳错（失衡）、阴刑阳德（刑罚与宽恕）、阴争阳扰（过度活跃导致混乱）等。

三、 奇门遁甲中的阴阳架构

奇门盘中处处皆阴阳，是断局的基础：

1. 盘面阴阳

  • 天盘与地盘：
    • 天盘（上）： 动、外显、阳。
    • 地盘（下）： 静、内隐、阴。
  • 内外盘（根据阴阳遁划分）：
    • 阳遁： 一、八、三、四宫为内盘；九、二、七、六宫为外盘。
    • 阴遁： 反之。

2. 用神阴阳（九星与八门）

  • 九星（天盘）：
    • 阳星： 天蓬、天任、天冲、天辅。
    • 阴星： 天英、天芮、天柱、天心。
    • 中性： 天禽星（寄宫于天芮，通常视为中性或随宫论）。
  • 八门（人盘）：
    • 阳门： 休、生、伤、杜。
    • 阴门： 景、死、惊、开。
    • 重要认知： 阳不代表绝对的“吉”，阴不代表绝对的“凶”。
      • 例：伤门属阳，但代表受伤、激烈；开门属阴，但代表敞开、沟通。

3. 干支阴阳

  • 十天干：
    • 阳干： 甲、丙、戊、庚、壬。
    • 阴干： 乙、丁、己、辛、癸。
  • 特殊概念：
    • 五阳时： 甲、乙、丙、丁、戊（用于占卜方法的划分）。
    • 五阴时： 己、庚、辛、壬、癸。

四、 阴阳在预测实战中的应用（核心心法）

1. 预测基础：十干克应与四象

利用天盘干与地盘干的阴阳组合（四象），判断事物的状态，不可死记硬背，需灵活取象：
四象符号（基于《周易》爻象，上为天盘，下为地盘）以及对应的实战克应总结：

  • 老阳（天阳 + 地阳）

    • 符号： ⚌ （上 — / 下 —）
    • 状态： 阳气过盛，亢奋，两强相遇。
    • 象意： 物极必反，如同两个壮汉互不相让，内部矛盾彻底爆发，发展到了尽头即将走向反面。
    • 案例（戊+庚）： 天盘甲子戊（阳）+ 地盘甲申庚（阳）。
      • 格局： “值符飞宫”，甲庚相冲。
      • 断语： 主换地方、由于内部矛盾激化导致的变动、关系破裂、惹不起躲得起。

  • 老阴（天阴 + 地阴）

    • 符号： ⚏ （上 - - / 下 - -）
    • 状态： 阴气过重，沉寂，甚至停止。
    • 象意： 事情发展到头，毫无生机，彼此冷漠，缺乏信任，或者陷入泥潭动弹不得。
    • 案例（辛+己）： 天盘甲午辛（阴）+ 地盘甲戌己（阴）。
      • 格局： “入狱自刑”，奴仆背主。
      • 断语： 主关系冰冷、有苦难言、诉讼难伸、自己给自己找罪受。

  • 少阴 / 少阳（阴阳混杂）

    • 符号： ⚍ （上 - - / 下 —） 或 ⚎ （上 — / 下 - -）
    • 状态： 相对和谐，但也存在消耗。
    • 象意：
      • 得位（天阳地阴 ⚎）： 天本为阳，地本为阴，这种组合符合自然之道（如丙+癸，虽五行相克，但阴阳位置得当），相对顺畅。
      • 失位（天阴地阳 ⚍）： 天盘为阴，地盘为阳。虽然阴阳配合，但这不仅违反了“天尊地卑”的自然属性，也如同“握手时暗中较劲”，表面和谐，内在有摩擦和损耗。

2. 核心警告：不可抛弃阴阳五行

  • 误区： 初学者容易迷失在九星、八门、格局的吉凶名词中，忘记最基础的阴阳五行生克。
  • 案例分析（丙+癸）：
    • 表象： 天盘丙（阳）+ 地盘癸（阴），阴阳搭配似乎和谐。
    • 本质（五行）： 丙为火，癸为水。水火不容，水火交战。
    • 结论： 虽阴阳相吸，但五行相克，本质是动荡和灾祸（“华盖悖师”，阴人害事）。
  • 心法： 阴阳平衡是理想状态。预测时，必须透过复杂的符号，回归到最底层的阴阳（性质/状态）和五行（生克/能量）来定性。

五、 总结

学习奇门遁甲，阴阳五行是“本”。无论盘面多么复杂，最终都要回归到阴阳的平衡与五行的生克上。所有的预测技巧（取象、读局）都是基于对这些基础概念的深刻理解与发散思维（远取诸物，近取诸身）。切忌铁口直断，需结合具体情境灵活变通。

《五行原论》核心知识精炼总结

一、 核心思维导图

  • 学习心法：术数（如奇门、六壬）是不同体系的融合。学习重在掌握思路与原理，而非死记硬背排列组合。
  • 五行本质：五行不仅仅是物质（金木水火土），而是宇宙间五种不同的运行状态（行 = 顺天行气）。

二、 五行的历史与演变

  1. 起源：最初被称为“五材”，指民生日常使用的五种物质资料。
  2. 哲学化（战国）：
    • 邹衍：提出“五德终始说”，将五行用于解释朝代更替和政治合法性（如秦属水德，汉属土德）。
  3. 宇宙观（汉代）：
    • 董仲舒/淮南子：构建了完整的五行宇宙图景，将五行与天文、医学、五音、五味、社会秩序等宏大视角结合，形成“天人感应”的基础。

三、 五行的定义与象意（出自《尚书·洪范》）

这是本课最核心的定义，五行即五种“气”的运行状态：

1. 水：曰“润下”

  • 运行状态：向下流动、渗透、滋养、潜藏。
  • 核心特质：湿润、寒冷、流动性（代表事物的不稳定、变化、发展）。
  • 对应象意：
    • 人物/性情：智慧（水主智）、胆大（如天蓬星）、好色、善变、目光远大。
    • 身体：肾脏、泌尿系统、耳朵。
    • 方位/季节：北方、冬天。
    • 颜色：黑色。

2. 火：曰“炎上”

  • 运行状态：向上升腾、扩散、发散。
  • 核心特质：火苗向上是虚的（不持久、三分钟热度），但在表面是光鲜亮丽、炽热的。
  • 对应象意：
    • 事物：文化、文明、电子产品（手机/电脑）、宣传广告、证件（合同/毕业证）、热闹场所（酒吧/夜市）。
    • 人物/性情：热情、礼貌（火主礼）、爱打扮、急躁、虚荣。
    • 身体：心脏、血液、眼睛。
    • 方位/季节：南方、夏天。
    • 颜色：红色、紫色。

3. 木：曰“曲直”

  • 运行状态：能屈能伸、舒展向上。与火的“虚”不同，木的向上是扎实的、生发的。
  • 核心特质：生长、条达、仁慈。
  • 对应象意：
    • 事物：教育、印刷、纸张、医药、种植。
    • 人物/性情：仁慈（木主仁）、身材修长（高/瘦）、白面书生感、倔强。
    • 身体：肝胆、四肢、毛发、神经系统。
    • 方位/季节：东方、春天。
    • 颜色：绿色、青色。

4. 金：曰“从革”

  • 运行状态：变革、肃杀、收敛、顺从与改变。
  • 核心特质：
    • 强金：变革、杀伐（刀剑、武器）、果断。
    • 弱金：装饰（首饰、珠宝）。
  • 对应象意：
    • 事物：金属、金融、汽车、改革（如辛亥革命）、决断。
    • 人物/性情：义气（金主义）、刚毅、冷酷、威严、棱角分明、皮肤白皙。
    • 身体：肺、骨骼、呼吸系统。
    • 方位/季节：西方、秋天。
    • 颜色：白色、金色。

5. 土：曰“稼穑”

  • 运行状态：承载、生化、转化、播种与收获。
  • 核心特质：土在五行中最为特殊，起承载和转换作用（长夏/四季土），连接万物的生发与收藏。
  • 对应象意：
    • 事物：房地产、建筑、丧葬、孕育、包容、迟缓（山止之意）。
    • 人物/性情：诚信（土主信）、敦厚、迟钝、固执、母亲般的包容。
    • 身体：脾胃、皮肤、肌肉。
    • 方位/季节：中央、四季末（季节转换期）。
    • 颜色：黄色。

四、 五行的生克逻辑（底层原理）

生克不是简单的“生”就是好，“克”就是坏，而是状态的“发散”与“收敛”。

1. 相生（状态的发散与演进）：

  • 水生木：“润下”滋养了“曲直”，使其得以发散生长。（如：安慰的话语浸润心灵，让人振作）。
  • 木生火：“曲直”发展到极致，向上升腾变为“炎上”（虚无、抽象化）。（如：事物发展到顶端变得抽象、哲学化）。
  • 火生土：“炎上”发展到极点，开始产生结果，进入“稼穑”（收获/转化）阶段。（如：抽象的流量转化为实际的粉丝/收益）。
  • 土生金：“稼穑”收获完成后，面临新的开始，引发“从革”（变革）。
  • 金生水：“从革”变革之后，重新衍生出“润下”（滋润、潜藏），开始新的循环。

2. 相克（状态的收敛与遏制）：

  • 水克火：“润下”遏制了“炎上”的虚幻与升腾，使其收敛。（不仅是水灭火，而是态势上的压制）。
  • 火克金：“炎上”的向上趋势遏制了“从革”的剧烈变革。（如：为了维持表面的繁荣/利益，暂时搁置激进的改革）。
  • 金克木：“从革”的肃杀打断了“曲直”的生长趋势。（如：强制改革改变了原有的发展方向）。
  • 木克土：“曲直”的生长打破了“稼穑”的稳定与收藏。（如：还在生长阶段，拒绝过早的收工/收获）。
  • 土克水：“稼穑”的阻滞/堤坝遏制了“润下”的流动。（如：停止探索，强制结算）。

3. 反克与强弱：

  • 克制关系受能量（旺衰）影响。如“土多金埋”、“水多土流”（大洪水冲垮堤坝）、“火旺水干”。

五、 旺衰体系（定性定量）

判断五行能量强弱的标准，主要依据月令（季节）：

  • 口诀：同我者旺，我生者相，生我者休，克我者囚，我克者死。
    • 旺（100%）：当令之气（如春天木旺）。
    • 相（次旺）：被当令者所生（如春天火相）。
    • 休（休息）：生当令者（如春天水休）。
    • 囚（被困）：克当令者（如春天金囚）。
    • 死（最弱）：被当令者克（如春天土死）。
  • 应用：根据旺衰调整策略。如自身处于“休囚死”地，不宜硬碰硬，宜采取策略（如“贪生忘克”）。

六、 运筹与化解（通关）

  • 原理：当五行相克造成危害时（如土克水），引入第三种五行进行“通关”。
  • 方法：寻找能连接两者的五行（A克B，找C，使A生C，C生B）。
    • 案例：土克水。
    • 解法：引入金。
    • 逻辑：土生金，金生水。金化泄了土的克性，转而生助水。
    • 实操：在风水或布局中放置属金的物品（如金属首饰、铜器）来化解土对水的克制。

七、 奇门遁甲应用初探

  • 九宫五行：坎（水）、离（火）、震/巽（木）、乾/兑（金）、坤/艮/中（土）。
  • 断局思路：
    • 日干：代表求测人。
    • 时干：代表所测之事。
    • 值使门：代表事情的最终结果。
    • 五行生克判断：看宫位、天干、星门的五行关系。例如，时干（事）克日干（人），事情难办；但若值使门（结果）状态好或有生助，则结果可能依然顺利。
    • 案例：虽然事情本身（五行相克）不顺（如火克金），但结果（值使门）若有吉格或相生，则过程曲折但结局可成。

天干冲合与核心体系总结

一、 核心基底：阴阳与五行

奇门遁甲无论读象还是技法，核心皆未脱离阴阳五行。

1. 阴阳的三个层面

  1. 属性划分： 最基础的分类（如甲为阳，乙为阴）。
  2. 整体宏观： 气的流转。从少阳 →\to→ 老阳 →\to→ 少阴 →\to→ 老阴。例如一年中阳气从冬至上升，至夏至极盛，随后阴气渐生。
  3. 组合碰撞： 阴阳之间的相互作用（如刻应）。

2. 五行的本质（重要）

五行不仅是物质，更是五种运行状态。必须理解其“形之所在”，而非死记硬背。

  • 生克比通： 熟练掌握生、克、比和（同属性）、通（制化通关，如木克土用火通关）。
  • 运行状态（本质）：
    • 木曰曲直： 生发、舒展。
    • 火曰炎上： 向上、虚无。
    • 水曰润下： 向下、滋润、受重力影响。
    • 金曰从革： 变革、肃杀。
    • 土爰稼穑： 长养、承载。
  • 相克原理举例（水克火）： 水是受重力向下润下，火是炎上飘散。二者运行轨迹背道而驰、撕裂。
    • 象意： 水克火在关系中代表观念不合、南辕北辙；在财运中代表条件不符、盲目求变、岗位不适。

二、 十天干的起源与架构

1. 体系融合

干支体系是阴阳（2）与五行（5）的融合（2×5=102 \times 5 = 102×5=10），并非单线衍生。

  • 天文起源说（重点）： 古人观测水星（辰星）。水星约10年绕天41圈回归原点（准周期），将其运行划分10个阶段，对应十天干。
  • 其他学说： 包含植物生长、人体器官、怀胎、房屋建筑等，多为后世附会，但有助理解象意。

2. 十天干阴阳本质论（核心心法）

不能简单将甲乙视为独立的木，而是同一种五行状态的阴阳两面。

  • 阳干（甲丙戊庚壬）： 代表事物的本质、性质定调。
  • 阴干（乙丁己辛癸）： 代表事物的表象、具体表现形式。
    • 例：甲木为阳性曲直（本质是新生、栋梁）；乙木为阴性曲直（表象是花草、弯曲、具体的形）。

3. 气数流转

  • 甲乙（少阳）→\to→ 丙丁（老阳）→\to→ 戊己（中央土转换）→\to→ 庚辛（少阴）→\to→ 壬癸（老阴）。
  • 奇门喜阳（木火通明），多主积极；畏阴（金水肃杀），多主消极（如庚格、天网）。

三、 十天干具体象意精炼

  1. 甲（阳木）： 首领、栋梁、性质为“真”。得时（旺）为栋梁，失令（衰）为废材。性格自负，受水生（壬癸）太过则漂泊无依。
  2. 乙（阴木）： 辅佐、花草、性质为“曲”。柔润、委婉、依附权势（不硬拼）。得时繁华，失令枯朽。
  3. 丙（阳火）： 太阳、权威、性质为“烈”。光明艳丽，急躁刚愎。乱子（麻烦）的象征。得令辉煌，失令灰槁（三分钟热度）。
  4. 丁（阴火）： 星烛、玉女、性质为“媚”。最吉之奇。灵性、洞察细节、善变。得时洞察奸邪，失令斤斤计较、暧昧难明。
  5. 戊（阳土）： 山岗、甲的替身（甲≈戊）、性质为“躁”。资本、信用、憨厚但粗鲁。得时豪雄，失令柔弱。
  6. 己（阴土）： 田园、策划、性质为“博”。私欲、陷阱、坟墓。柔顺坦贞，但内藏贪欲。
  7. 庚（阳金）： 兵器、阻碍、性质为“坚”。白虎之性，刚硬、杀伐。可柔化（乙合庚），不可刚制（火克庚反激其怒）。
  8. 辛（阴金）： 珠宝、错误、性质为“秀”。罪人、变革。得时如黄钟，失令如瓦缶。
  9. 壬（阳水）： 江河、流动、性质为“润”。天牢、变动、迷茫。代表运输、物流、无拘束。
  10. 癸（阴水）： 雨露、罗网、性质为“浊”。阴暗、困顿、杂质、欲望。代表终结、小腿足部、不洁之液。

四、 天干五合（重点技法）

源于“五运经天”的天文运气学说。合化后的五行在断局中往往占据主导地位。那么所谓合化了，就是它们俩聚在一起，类似于一个化学反应一样，能够合化出一个新的五行。

  1. 甲己合化土 —— 中正之合
    • 象意： 尊卑有序，名利双收。
    • 内涵： 甲（领导）得名声，己（策划/卑微）得实利。甲木克己土，名义上压制，实际上利益归于土。
  2. 乙庚合化金 —— 仁义之合
    • 象意： 刚柔兼济，但也主虚伪、面和心不和。
    • 内涵： 金为刚，木为柔。为了达成目的（化金），乙木委曲求全，更金收敛杀气。名声受损，有虚诈之嫌。
  3. 丙辛合化水 —— 威制之合
    • 象意： 表面和谐，内耗严重。
    • 内涵： 丙为阳火，辛为阴金，合化成水反克丙火。为了维持表面的顺利（合），实际上耗损了自身的元气（丙火受损）。主前程婚姻顺，但官运内耗。
  4. 丁壬合化木 —— 淫特之合（非单纯淫荡）
    • 象意： 欲望强烈，甚至无节制，但能生财/解难。
    • 内涵： “淫”指无节制（壬水漫流），“特”指私情/欲望。丁火为壬水之财，合化木又是丁之印。虽有欲望过度的负面，但能宣泄水势，通过欲望或私情达成转化，利于求财脱困。
  5. 戊癸合化火 —— 无情之合
    • 象意： 利益交换，如胶似漆但本质冷漠。
    • 内涵： 戊为老夫，癸为少妇。因“火”（利益/温暖）而聚。利在则合，利尽则散，本质是各取所需。

五、 天干四冲

  • 组合： 甲庚冲、乙辛冲、丙壬冲、丁癸冲。（戊己居中无冲）。
  • 象意：
    • 离散： 事情被打散。
    • 动态/改变： 凡事未发生遇冲则起，已发生遇冲则散。
    • 不稳定： 动荡、晃动。

六、 天干旺衰（十二长生）

基于火土同宫（火为夏，土为长夏）理论，反映天干在不同时间/宫位的能量状态。

1. 状态曲线（三角函数式）

  • 上升期： 长生（出生） →\to→ 沐浴（桃花/败地/有心无力） →\to→ 冠带 →\to→ 临官（禄地/最好状态，前途无量）。
  • 顶峰与衰退： 帝旺（顶点，物极必反） →\to→ 衰 →\to→ 病 →\to→ 死 →\to→ 墓（入库/被困） →\to→ 绝 →\to→ 胎 →\to→ 养。

2. 推导规律（阳顺阴逆）

  • 阳干顺行： 甲长生在亥，顺时针数。
  • 阴干逆行： 乙长生在午（甲之死地），逆时针数。
  • 口诀规律： 阳死阴生，阴死阳生。
    • 例：甲（阳木）长生在亥，死在午；乙（阴木）则长生在午，死在亥。
  • 速查（阳干长生位）：
    • 甲 →\to→ 亥
    • 丙/戊 →\to→ 寅 （火土同宫）
    • 庚 →\to→ 巳
    • 壬 →\to→ 申

七、 合五行（高阶初论）

除了正五行（甲木、丙火等），还存在由干支组合产生的“合五行”（如丁壬化木，这个“木”即合五行）。

  1. 相互依存与制约： 合化出的五行（如木）虽由原五行（水、火）生成，但带有原五行的杂气，非纯气。
  2. 主导作用： 在断局中，若出现合化，应以合化后的五行为主导，原五行退居次要地位（微调/背景）。
    • 例：戊癸合化火。断身体时，土（戊）因生火而增强，水（癸）因克火而耗损。导致土强水弱，可能引发消化系统（土）亢进或肾脏/血液（水）受损的疾病。
  3. 过程与结果： 正五行是过程，合五行是最终导向。
  4. 地支三合局应用： 如申子辰三合水局，求财不能只看“子”方，必须经过“申”和“辰”的过渡或牵制才能成事。

地支辨析与实战应用总结

一、 地支的起源与象意

地支不仅是计数符号，更包含了天文学、植物生长及阴阳消长的底层逻辑。

1. 起源假说

  • 神话与天文： 虽有“天皇地皇”及“常羲生十二月”之说，但更具学术价值的是植物生长说（借字表意）与天文学（岁差与观测）。
  • 现代观点： 甲骨文分析显示干支间存在互相孕育的关系（如癸孕甲），但这属于现代推论。

2. 植物生长学说（十二地支核心象意）

地支描述了万物从萌芽到衰亡的全过程：

  • 子（坎宫）： 阳气始萌，万物在地下滋生。象意： 阴极之水、雨露、流动、润下，外阳内阴。
  • 丑（艮宫）： 纽也，阳气初升，被束缚待发。象意： 金库、湿土、随和、缓慢。
  • 寅（艮宫）： 演也，万物始生，破土而出。象意： 粗壮大树、爆发力、启动。
  • 卯（震宫）： 冒也，万物冒地而出，门户。象意： 嫩芽、门户、细致、波折。
  • 辰（巽宫）： 伸也，万物舒展。象意： 水库、湿土、杂气、易变。
  • 巳（巽宫）： 已也，万物已成。象意： 变色龙、外强中干、阴火。
  • 午（离宫）： 忤也，阴阳交融，盛极将衰。象意： 烽火、燥热、善变、只有三分钟热度。
  • 未（坤宫）： 味也，果实成熟有滋味。象意： 木库、燥土、也是花园。
  • 申（坤宫）： 身也，身体成型。象意： 传送、坚硬、破坏力、肃杀。
  • 酉（兑宫）： 秀也，老也，成熟至极。象意： 阴金、精巧、又名从魁。
  • 戌（乾宫）： 灭也，万物灭藏。象意： 火库、燥土、欺诈、虚伪。
  • 亥（乾宫）： 核也，万物收藏成种子。象意： 悬河、天门、好静但势猛、五湖四海。

二、 地支的作用关系（核心重头戏）

1. 地支六合（原理：日月合朔）

  • 原理： 源于天文上的“月将”（太阳位置）与“月建”（斗柄指向）的对应。
  • 合化存疑： 奇门中不主张地支六合有合化（如子丑化土）。合化理论源自《七政四余》的星象对应，不应生搬硬套至奇门。
  • 具体象意：
    • 子丑合： 泥合/逆合。虽合作但有不情愿的付出或损失。
    • 寅亥合： 破合。同心协力但过程有破耗，先合后破。
    • 卯戌合： 淫合/无自控。随波逐流，意志不坚定，缺乏主见。
    • 辰酉合： 暗合。私下约定，若操作不当易生官非。
    • 巳申合： 刑合。吉凶参半，似进似退，纠缠拉扯。
    • 午未合： 阴阳之合。中正，较为平稳。

2. 地支三合（原理：时间/岁差）

  • 原理： 基于地球公转与自转的岁差（每四年一闰的刻度变化），在时间维度上同气相求。
  • 局象： 申子辰合水，亥卯未合木，寅午戌合火，巳酉丑合金。
  • 状态： 包含长生、帝旺、墓库三个状态，体现五行能量的稳定与循环。
  • 半合： 缺一字为半合，通常帝旺（中神）必须存在才有力。

3. 地支三会（原理：空间/方位）

  • 原理： 空间维度的能量汇聚。
  • 局象： 寅卯辰会东方木，巳午未会南方火，申酉戌会西方金，亥子丑会北方水。

4. 地支相刑（原理：三合与三会的冲突）

  • 本质： 是“制约”与“规劝”，而非单纯的殴打。源于占星学中的90度相位（四分相），代表压抑与冲突。
  • 推导逻辑： 金刚火强自行其方（三合与三会同位），木落归本水流趋东（位置互换）。
  • 分类与象意：
    • 无礼之刑（子卯）： 不懂察言观色，混乱，无礼数（如母子反目）。
    • 无恩之刑（寅巳申）： 忘恩负义，以德报怨，阻碍多，损失重。
    • 恃势之刑（丑戌未）： 仗势欺人，过于霸道反招阻碍，得不到的骚动。
    • 自刑（辰午酉亥）：
      • 辰辰： 怀才不遇，悲观抑郁。
      • 午午： 脾气暴躁，精神抑郁。
      • 酉酉： 冲动，丧失理智。
      • 亥亥： 欲望过重，纵欲之象。

5. 地支相冲（原理：对冲）

  • 关系： 子午、丑未、寅申、卯酉、辰戌、巳亥。
  • 本质： 180度对冲，代表变动、冲突、散开。

三、 人元司令与地支藏干

藏干的本质是气的流通与交接（上月余气 -> 下季孕育 -> 本月旺气）。

1. 藏干推导逻辑（以寅月为例）

  • 常规误区： 认为是“甲丙戊”。
  • 正确气流： 戊 -> 丙 -> 甲。
    • 戊（余气）： 承接上个月（丑月）的土气转化。
    • 丙（中气）： 孕育下一个季节（夏天火）的气。
    • 甲（本气）： 春天木的本气。
  • 特殊规律：
    • 四正（子午卯酉）： 气专，多只藏本气（午火特殊，藏丁己，因火土同宫）。
    • 四生（寅申巳亥）： 气杂，藏本气、中气（长生之气）、余气。
    • 四墓（辰戌丑未）： 藏本气（土）、库气（入墓之五行）、余气。

四、 合五行全论（实战应用）

虽然理论上存疑，但在实战预测中，结合合化五行可判断吉凶趋势。

1. 合化水（丙辛、巳申、申子辰）

  • 六合水/丙辛： 刚硬，短期有利，长期不利。
  • 三合水（申子辰）： 土中金水，利于求财、求学、印星（庇护），不利打官司（官杀）。

2. 合化木（寅亥、丁壬、亥卯未）

  • 寅亥（六合）： 需外力帮助，但外力不可靠/短暂。
  • 丁壬： 过程矛盾重重，需有技术且坚持不懈才能成功。
  • 三合木（亥卯未）： 土木相滋，利于名气、官职，但前提是“自身技术/条件过硬”。

3. 合化火（卯戌、戊癸、寅午戌）

  • 卯戌（六合）： 需具备“技术（木）”与“人脉（土）”双重条件，缺一不可。
  • 戊癸： 虚火。利于广告、虚名、荣誉，不利求实利/钱财。
  • 三合火（寅午戌）： 自然之火，反而是利于求财、合作及婚姻的。

4. 合化金（辰酉、乙庚、巳酉丑）

  • 辰酉（六合）： 土多金埋，易怀才不遇，无平台。
  • 乙庚： 仁义之合，利于征战、刚猛之事。
  • 三合金（巳酉丑）： 利于婚姻，合作伙伴多为女性。

5. 合化土（子丑、甲己）

  • 子丑（六合）： 受水制约，阴阳失衡。做事需当机立断，否则错失良机。
  • 甲己： 若木有生机且不妄克土，则大利聚财，利益丰厚。

总结建议： 奇门遁甲是模型学，技法（如地支刑冲）是填充模型的血肉。初学者应先掌握模型（九宫、节气、排盘），再精研技法（地支微观关系），方能达到“看山又是山”的境界。
