个人开发者上架Steam游戏全流程指南

发表于2025-10-29|更新于2025-10-29|其他实用工具
|总字数:1.1k|阅读时长:3分钟|浏览量:|评论数:

前言

记录下Steam上架步骤和坑点（我发现了，凡是类似于SDK，注册信息一类的流程，就没有不踩坑的，真是奇了怪哉
完整流程：

设置 Steamworks 帐户

https://partner.steamgames.com/
入门第一步是注册成为 Steamworks 合作伙伴。也可以使用自己的Steam账号直接登录，选择注册开发者账号
一旦您设置好您的帐户，并且银行、税务和公司信息已受验证，您就可以添加更多用户并开始设置您的第一款产品，或“应用程序”。

注册账户与税务信息注意点

整个注册过程中只有这里有坑，但是这里的坑也最多，流程上是没什么问题，指引的很好，就是细节要注意
通用的，这个流程所有的信息都需要用英文填写，不论是姓名，地址，银行信息，全都是

  • 自己的姓名，必须按顺序全部大写，例如：张三，就需要填ZHANG SAN，空格加不加都行
  • 地址，填写身份证上的地址，注意需要使用谷歌翻译翻译成英文，例如：深圳市南山区深南大道9037号，就需要填9037 Shennan Avenue, Nanshan District, Shenzhen
    • 具体的州，镇，省其实随便对应下就行，只要保证纯英文，且和你身份证上住址一致即可
    • 印象中有个选项，勾选后确认这个地址即使不合法自己也担保是存在的，因为老外的地址写法和我们不一样
  • 银行卡信息，所有信息（SWIFT代码，地址信息等）都可以直接联系银行柜台让他发你，注意要开通外汇功能，用来接收美元收益，我有张香港银行卡，直接通过了，这个阶段倒没踩坑
  • 提交审核后，过不了几分钟他就会发你两封邮件，各自让你上传两张图片
    • 身份证：正反面照片两张
    • 身份认证：自己手持身份证正反面的照片两张
      • 注意，需要在身份认证的两张照片上添加自己英文名翻译（例如上面提及的ZHANG SAN），来让审核机构确认，随便搞个P图软件放个醒目的文本在旁边即可
        • 如果被打回，就把地址也翻译了贴旁边
        • 如果一直被打回，直接点底部的Steam客服协助即可，全程人工服务，肯定搞得定，就是有时差，往往一条信息发过去要等一天才有回复

下载 Steamworks SDK

完成登记流程后，需要下载 Steamworks SDK，其中包括在 Steam 上创建和上传产品所需的所有脚本和模板。
您可以在此处下载 Steamworks SDK 的最新版本并在 Steamworks SDK 文档中学习到更多关于 SDK 的内容。

创建 Steam 生成版本与 depot

要让您的应用程序在 Steam 上顺利运作，您需要上传一个生成版本并配置一些设置。 此过程在上传至 Steam 中有详细介绍。 至于何时创建并测试您的应用程序，您可自行决定。

配置 Steamworks 功能

您可以自由选择要使用多少 Steamworks 功能。 参见功能，了解更多信息。

创建您的商店页面及定价

您一边努力使应用程序在 Steam 上顺利运作的同时，另一边也可以着手准备游戏或软件的商店状态。 如果您在此过程中有任何问题，可以在商店页面/市场营销论坛中提出问题或寻求帮助。 参见商店状态，了解更多信息。
这里倒没什么坑，就是很麻烦，他要求的图片分辨率必须分毫不差才行，耐心点
需要注意的是，必须在应用管理界面把所有的可点击项一个个点进去，然后填充内容，我就因为漏点了 内容调查 导致初次审核被卡了

准备发行

大部分在 Steam 上发行产品的准备工作都可以借助工具和文档自行完成。 但是在您即将发行产品之前，Valve 需要审核您的商店页面、建议定价和游戏生成版本。
参见审核流程，了解详情。
